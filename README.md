# 장상 네트워크 추론 게임

장상론을 장부 암기가 아니라 기능 네트워크 추론으로 익히는 교육용 웹게임입니다.

## 게임 목적

증상과 추가 단서를 바탕으로
주 장부
연계 장부
장부 관계
병기
를 추론하면서 장상론의 관계적 사고를 훈련합니다.

## 게임 방법

1. 화면에 제시된 증상과 추가 단서를 읽습니다.
2. 주 장부를 선택합니다.
3. 연계 장부를 선택합니다.
4. 관련 부를 선택합니다.
5. 장부 관계를 선택합니다.
6. 병기를 선택합니다.
7. 짧은 서술칸에 근거를 한 줄로 적습니다.
8. 제출 버튼을 눌러 점수와 해설을 확인합니다.
9. 다음 문제 버튼을 눌러 다음 문항으로 진행합니다.

## 점수 방식

주 장부 1점
연계 장부 1점
관계 2점
병기 2점
짧은 서술 1점

총 7점 만점입니다.

## 학습 포인트

이 게임은 정답만 맞히는 게임이 아닙니다.
증상에서 장부 기능을 추론하고
장부 간 관계를 설명하는 과정이 중요합니다.

## 권장 학습 순서

하 난이도부터 시작합니다.
그다음 중 난이도로 장부 관계를 익힙니다.
마지막으로 상 난이도에서 복합 추론을 연습합니다.
